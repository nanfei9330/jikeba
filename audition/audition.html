HTML简介和发展史

HTML是超文本标记语言，经历了2.0版，到3.2版、4.0版、4.01版，然后经历XHTML1.0再到目前HTML5。

HTML5有哪些新特性？

  • 使用简洁的DOCTYPE
  • 增加了更加语义化的标签，如： <header>、<footer>、<nav>
  • 表单增强，提供了更多input类型，如：color、email、date、time
  • 对本地离线储存有更好的支持
  • 新增用于绘画的canvas标签
  • 新增用于媒介回放的video和audio元素

CSS3有哪些新特性？

  • 实现圆角
  • 阴影特效
  • 媒体查询
  • 动画，2D转换
  • 弹性盒模型
  • 增加伪类选择器

对WEB标准以及W3C的理解与认识

就是写代码更加规范，比如标签要闭合、标签使用小写、不乱嵌套、结构与表现分离、使用外链css和js，目的是使用更少的代码，更容易维护，内容能被更广泛的设备访问，提高网站易用性。

Doctype有什么作用? 严格模式与混杂模式如何区分？它们有何意义?

浏览器根据DOCTYPE选择用什么文档类型规范来渲染这个文档，严格模式是浏览器根据规范去显示页面。混杂模式是一种比较宽松的向后兼容的方式显示页面。如果没有声明，那么默认为混杂模式。

link和@import的区别

link，先加载CSS，后加载HTML，无兼容性，支持使用js改变样式

@import，先加载HTML，后加载CSS，在css2.1以下浏览器不支持

css reset的作用

因为浏览器的兼容问题，不同浏览器对有些标签的默认值是不同的，如果没对CSS初始化往往会出现浏览器之间的页面显示差异，通过reset重置浏览器的css默认属性，让它们统一

对闭包的理解

闭包是可以访问另一个函数作用域的函数。如果需要在函数外部调用函数内部变量，那就需要用到闭包，但是闭包会使得函数的变量保存在内存中，使得内存消耗大，使用不当很容易造成内存泄露


//简单例子：getscope函数就是一个闭包
function checkscope(){
    var a=1;
    function getscope(){
        a++;
        return a;
    }
    return getscope;
}
alert(checkscope()());

cookie 和session 的区别

cookie数据存放在客户的浏览器上，session数据放在服务器上，考虑到安全应当使用session。考虑到减轻服务器性能方面，则应当使用COOKIE。

对JSON的了解?

JSON是一种数据交换格式，易于读写, 占用带宽小,跟XML和HTML相比更简单更灵活。

{'age':'12', 'name':'back'}

null和undefined的区别？

null是一个表示"无"的对象，转为数值时为0；undefined是一个表示"无"的原始值，转为数值时为NaN。
